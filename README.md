# SkinPath-CNN

## Setup Environment - Anaconda
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
```

## Setup Environment - Shell/Terminal
```
mkdir project_cnn
cd project_cnn
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run streamlit app
```
streamlit run app.py
```
